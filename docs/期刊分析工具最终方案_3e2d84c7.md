## 一、数据表设计（4 张表）

### 1.1 journals - 期刊主表（含费用+收录状态）

```
_id                    自动生成
openalex_id            string (唯一)    OpenAlex 期刊 ID（如 S1234567890）
title                  string           期刊名称
title_abbrev           string           期刊缩写（alternate_names[0]）
issn_print             string           印刷版 ISSN（从 issns 数组中提取 print）
issn_online            string           电子版 ISSN（从 issns 数组中提取 electronic）
issn_l                 string           Linking ISSN
homepage_url           string           官网链接（homepage_url）
doi_prefix             string           DOI 前缀（从 works 中推断）
publisher              string           出版社（publisher）
country                string           出版国家（DOAJ补充）
language               string           出版语言（DOAJ补充）
launch_year            number           创刊年份（手工维护/推断）
submission_url         string           投稿地址（DOAJ补充）
type                   string           journal / conference proceedings / repository（type）
peer_review_model      string           审稿模式（手工维护）
status                 string           active / ceased / transferred（is_active）

// ---- OA状态（OpenAlex直接提供） ----
is_open_access         boolean          是否OA（is_oa）
oa_type                string           gold / hybrid / green / bronze（需从works推断）
doaj_seal              boolean          DOAJ Seal（DOAJ补充）

// ---- 收录状态（仅免费数据源可获取的） ----
is_in_doaj             boolean          DOAJ收录（DOAJ API判断）
has_pmid               boolean          PubMed收录（NCBI搜索判断）
has_pmc                boolean          PMC收录（NCBI搜索判断）
pmid_count             number           PubMed文章数（NCBI返回count）
pmc_count              number           PMC文章数（NCBI返回count）

// ---- 核心指标快照（从OpenAlex summary_stats提取最新值） ----
works_count_latest     number           总发文量（works_count）
cited_by_count_latest  number           总被引量（cited_by_count）
h_index_latest         number           h指数（summary_stats.h_index）⭐
i10_index_latest       number           i10指数（summary_stats.i10_index）⭐
two_year_mean_citedness_latest  number  2年平均引用率（summary_stats.2yr_mean_citedness）⭐

// ---- 学科分类快照（从topics提取Top5） ----
top_topics             array            Top5学科 [{display_name, count}] ⭐

created_at             date
updated_at             date
last_synced_at         date             最后同步时间
```

### 1.2 journal_subject - 学科分类表（一对多，独立表）

```
_id              自动生成
journal_id       string           关联期刊 ID
source           string           openalex / doaj
subject_level    number           1=Domain, 2=Field, 3=Subfield, 4=Topic
subject_id       string           OpenAlex Topic ID
subject_name     string           分类名称
parent_id        string           父级 ID
created_at       date
```

### 1.3 journal_metrics_yearly - 年度指标表

```
_id                        自动生成
journal_id                 string
metric_year                number

// ---- 自动获取（OpenAlex summary_stats + counts_by_year） ----
works_count                number           发文量（works_count）
cited_by_count             number           被引次数（cited_by_count）
h_index                    number           H-index（summary_stats.h_index）⭐
i10_index                  number           i10指数（summary_stats.i10_index）⭐
two_year_mean_citedness    number           2年平均引用率（summary_stats.2yr_mean_citedness）⭐

// ---- 年度统计（counts_by_year按年展开） ----
year                       number           年份
works_count_yearly         number           当年发文量
cited_by_count_yearly      number           当年被引量

// ---- 文章类型分布（从works聚合） ----
research_articles          number           研究论文数（type:article）
review_articles            number           综述数（type:review）
editorial_articles         number           社论数（type:editorial）
other_articles             number           其他类型

// ---- 作者画像（从works聚合Top N） ----
author_count               number           去重作者数
top_authors                object           [{name, count}] ⭐
top_institutions           object           [{name, count}] 
top_countries              object           [{name, count}] ⭐

// ---- 学科主题分布（x_concepts或topics聚合） ----
top_topics                 object           [{display_name, count}] ⭐

// ---- 手工维护（二期功能） ----
sjr                        number           SJR（SCImago网站被Cloudflare保护无法爬取）
snip                       number           SNIP（同上）
citescore                  number           CiteScore（同上）
quartile                   string           Q1/Q2/Q3/Q4（同上）
jif                        number           影响因子 (JCR付费)

created_at                 date
updated_at                 date
```

### 1.4 journal_warnings - 预警期刊表

```
_id              自动生成
journal_id       string
warning_level    string           high / medium / low
reason           string           预警原因
source           string           cabells / beall / cas / manual
status           string           active / resolved
created_at       date
updated_at       date
```

## 二、数据来源与获取方式

| 数据 | 来源 | 方式 |
|------|------|------|
| 期刊基础信息 | OpenAlex sources API | 自动（title, issns, homepage_url, publisher, type, is_oa, is_active） |
| **核心指标** | OpenAlex summary_stats | 自动（**works_count, cited_by_count, h_index, i10_index, 2yr_mean_citedness**）⭐ |
| **年度趋势** | OpenAlex counts_by_year | 自动（按年展开 works_count, cited_by_count）⭐ |
| **文章类型分布** | OpenAlex works API | 自动（聚合 type:article/review/editorial 数量）⭐ |
| **作者画像** | OpenAlex works API | 自动（聚合 top authors/institutions/countries）⭐ |
| **学科主题** | OpenAlex topics/x_concepts | 自动（聚合 Top5 topics 及计数）⭐ |
| OA/APC/License/国家/语言 | DOAJ API | 自动（补充OA详细信息） |
| PubMed/PMC收录 | NCBI E-utilities | 自动（按期刊名搜索判断） |
| DOAJ收录 | DOAJ API | 自动（判断是否在返回结果中） |
| JIF/SJR/SNIP/CiteScore/分区 | - | **二期手工维护**（SCImago网站被Cloudflare保护无法爬取） |
| WoS/Scopus/EI/CAS收录 | - | **二期手工维护**（付费数据库） |
| 预警数据 | - | 手工维护 |

## 八、重要说明

### 数据来源限制
- **SCImago 网站已被 Cloudflare 保护**，无法通过简单爬虫获取 SJR/SNIP/CiteScore/分区数据
- **WoS/Scopus/EI/中科院均为付费数据库**，无法免费获取收录状态和JIF
- **MVP阶段仅使用免费数据源**：Crossref + DOAJ + NCBI + OpenAlex

### 二期扩展建议
如需完整指标体系，建议二期增加：
1. `journal_metrics_manual` 表 - 由管理员手工录入 JIF/SJR/分区/h指数等
2. 后台管理界面 - 用于批量导入和维护付费指标数据
