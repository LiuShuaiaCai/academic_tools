## 搜索重构：服务端模糊搜索 + ID 精确定位
- 发现列表页搜索只搜本地已加载数据（云数据库 .get() 默认最多返回20条），超出的搜不到
- 方案：1）首页跳转传 targetId + targetTitle（title 仅填充搜索栏，定位靠 ID）；2）搜索走服务端 db.RegExp 模糊查询
- 改了 4 个文件：
  - home.js：goToItem 传 `targetId` + `targetTitle` + `autoEdit`
  - submissions.js：loadList 用 db.RegExp 搜索（title/journal/coauthors/tags），新增 locateById()
  - reviews.js：同上（paperTitle/journal），新增 locateById() 和 onSearch()
  - conferences.js：同上（name/shortName/location），新增 locateById()，onSearch() 走 loadList()
- applyFilter 不再重复做关键词过滤，只做状态/高级筛选
- submissions 的 onReachBottom 搜索时也支持翻页
